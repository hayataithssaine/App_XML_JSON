# App_XML_JSON
Application JavaFX de conversion bidirectionnelle entre XML et JSON. Elle propose deux méthodes : conversion automatique avec l'API Jackson et conversion manuelle avec algorithmes de parsing personnalisés. Interface graphique moderne avec zones de texte et boutons de conversion
